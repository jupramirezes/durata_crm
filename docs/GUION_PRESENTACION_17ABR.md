# Guión presentación DURATA CRM — 17 abril 2026

## Flujo de demo sugerido (20-25 min)

### 1. Contexto (2 min)
- Excel de 45MB, solo Sebastián lo entiende
- Sin seguimiento post-envío
- 74% de cotizaciones se pierden sin saber por qué
- Este sistema: CRM+Cotizador, una fuente, visibilidad real, APU automático

### 2. Dashboard (3 min)
- 5 KPIs: oportunidades activas, valor pipeline, cotizaciones mes, tasa cierre cant/valor
- 184 cotizaciones >7 días sin respuesta
- Pipeline activo clickeable

### 3. Pipeline Kanban (3 min)
- 7 columnas con # COT en cada card
- Dropdown "Orden" (valor, fecha, empresa, cotizador, # COT)
- Filtro por cotizador
- Drag & drop → adjudicar con valor + fecha

### 4. Oportunidad (5 min)
- Sidebar con empresa, contacto, etapa, valor
- Cotizaciones con APU + PDF adjuntos (click descarga)
- Recotizar → versión A automática
- Duplicar → crear nueva oportunidad
- Configurador: 33 productos con APU automático

### 5. PDF (3 min)
- Modal con condiciones por tabs
- PDF profesional: logo, datos, tabla, condiciones
- Descripción completa del producto

### 6. Data migrada (2 min)
- 5189 oportunidades, 5177 cotizaciones
- 354 APU + 379 PDF ya adjuntos
- Empresas clasificadas por sector
- Listo para usar HOY

### 7. Roadmap (2 min)
- Imagen IA del producto
- Chat Dashboard
- Notificaciones automáticas
- Vista productos por familia

### 8. Siguiente paso (1 min)
- Empezar a usar mañana para cotizaciones nuevas
- Preguntar qué falta
